HTML

  • The head element contains info about the webpage.
  • The body element represents the visible content shown to the user
  • The li element wraps each list item
  • ul stands for unordered list and ol stands for unordered list
  • HTML is a markup language and is the foundation of webpage development
  • The id and class are different but they are both used to idetify a specific element
  • Semantic HTML is important for code to be read easier as well as useful for those that use assistive technologies like screen readers

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is used to style a webpage
  • A margin is how much space we want around the outside of an element
  • Padding is how much space we want around the content inside an element
  • class selectors start with a . followed by the class name

Git

  • Git is a console that is used to manage files and connect them to github
  • Github is useful for version control as well as collaboration
  • cd stands for change directory and is used to navigate through folders
  • mkdir stands for make directory and is used to create a new folder within the current directory
  • code . will open the current directory in your code editor
  • git status checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
